Getting Started With LexPad

This section will walk you through the process of installing LexPad and creating your first project. Alternatively, you can download one of these sample projects to try out the software with an existing lexicon.

  1. Sample project: Wikchamni
  2. Sample project: Chukchansi
  3. Sample project: Spanish

Installing LexPad

Your OS will warn you about installing untrusted software. Signing costs $700/yr, which is expensive for a free project in its beta phase. The installation instructions explain how to mark LexPad as trusted and run it anyways.

Never do this for software unless you know you can trust the author!

Installing LexPad on Windows

  1. Download the Windows installer here if you haven't already.
  2. Double-click the installer and follow the onscreen prompts.
  3. If a desktop shortcut was created during installation, right-click it and select "Open file location", then find LexPad.exe. Otherwise, type "%appdata%" in the address bar of File Explorer and navigate to AppData/Local/LexPad. Open the folder with a name like "app-0.0.123" and find LexPad.exe.
  4. Right-click LexPad.exe and select "Properties". The properties window should have a checkbox near the bottom labeled "Unblock". Click it, then click "Apply" to mark the app as trusted.
  5. Success! You should now be able to open the app normally.

If a desktop shortcut wasn't created during installation, you can create one yourself. Hold shift and right-click LexPad.exe, then select "Create a shortcut".

Installing LexPad on Macintosh

These instructions were adapted from Apple's official help page.

  1. Download the Mac package file here if you haven't already.
  2. Double-click the package file and follow the onscreen prompts.
  3. After LexPad is installed, open it once. You should get a warning about untrusted software. Close the warning.
  4. Open the Apple menu and select "System Settings", then click "Privacy and Security" in the sidebar of the window.
  5. Find LexPad in the "Security" section. Click "Open", then click "Open Anyway". Enter your login password to mark the app as trusted.
  6. Success! You should now be able to open the app normally.

Creating Your First Lexicon

Creating a New Project

  1. First, create a folder to store your project in.
  2. Open LexPad, then click "New Project" in the welcome screen or the app menu.
  3. Select the directory you created, then click "Create".
  4. Success! Your new project will open automatically.

You can rename your project file, but make sure to save it and close LexPad before doing so.

Placing multiple projects in the same folder causes them to share their media assets. This can be helpful for studying subsets of your data or storing backup copies, but will make it hard to keep track of things if you mix lexicons from multiple languages.

Creating an New Entry

  1. Switch to the Lexicon Tab and click "New Entry" in the bottom left corner. Select the entry's catg / part of speech from the dropdown, or "Create New Catg".
  2. L1 words (generally english) are entered in the header. You can include multiple synonyms or spelling variations by separating them with semicolons. For example, typing "run; sprint" will add both "run" and "sprint" to L1. Similarly, typing "color; colour" will add both spelling variations.
  3. Add L2 word data to the "Wordforms" section. If words of this catg have more than one form (grammatical case, conjugations, etc), use the dropdown menu to select one. This section also supports synonyms and spelling variations, with the same semicolon notation.
  4. The "Sentences" section holds sentences, phrases, and other examples of L2 data. Use the "Eng" field for gloss, translation, etc.
  5. Only generic notes are currently supported, but multiple types will eventually be added. For now, type labels (ie "Anthropology: ...") in your notes so they can be auto-labeled later.
  6. Any number of images can be attached to entries. Click the image square in the header to open the image manager.
  7. Any number of audio can be attached to each wordform and each sentence. Click the plus icon to attach a file, then click the audio icon to play it. You can also use the three-dots menu on the right to open the audio manager.

Since each part of speech may have a different set of wordforms associated with it, case/form labels will be lost when an entry's catg is changed.
